### [不能落后，好好缕缕CSS滚动动画](https://www.zhangxinxu.com/wordpress/2024/08/css-scroll-timeline/)

```css
section {
    --flag: false;
    animation: setFlag 1ms;
    scroll-timeline: --detectScroll;
    animation-timeline: --detectScroll;
}

@keyframes setFlag {

    from,
    to {
        --flag: true;
    }
}

@container style(--flag: true) {

    /* 容器溢出 */
    button {
        display: block;
    }
}
```

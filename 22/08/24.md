### [navigator.sendBeacon](https://zhuanlan.zhihu.com/p/381796039)

> 关闭页面时，发送请求

```js
const sendBeacon = (url, data = {}) => {
    const blob = new Blob([JSON.stringify(data)], {
        type: 'application/json; charset=UTF-8',
    });
    return navigator.sendBeacon(url, blob);
};
```
